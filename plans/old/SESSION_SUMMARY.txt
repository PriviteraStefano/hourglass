ARCHITECTURE DECISIONS (ALL STRESS-TESTED)
================================================================================

✓ Approval Authority (No Ambiguity)
  WG Manager → approves time entries
  Org Manager → reallocates post-approval
  Finance → approves expenses, overrides locked entries

✓ Financial Period Cutoff (Compliance-Ready)
  Org sets cutoff date, entries lock, Finance can override with reason

✓ Time Entry Status (Simplified)
  Draft → Submitted → Approved (3 states only)

✓ Unit Binding (Flexible)
  WG config controls: enforce or allow user selection

✓ Audit Trail (Immutable)
  Single shared table, append-only, tracks all actions

✓ Org Hierarchy (Unlimited Nesting)
  No depth limit, recursive graph, SurrealDB relations
